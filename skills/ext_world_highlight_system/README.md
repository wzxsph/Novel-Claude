# 世界观名词高亮提示插件 (WorldHighlightSystem)

## 功能说明
这是一个小说创作辅助插件，能够在小说文本中自动识别并高亮显示特殊的世界观名词（如阵营、功法、特殊物品等），通过在名词前后添加【】框来突出显示。

## 主要特性
- **自动高亮**：在LLM生成文本前，自动将预设的特殊名词用【】框包裹
- **分类管理**：支持按类别（阵营、功法、特殊物品）管理高亮名词
- **动态更新**：可以通过工具接口动态添加、删除和查看高亮名词
- **灵活配置**：初始包含常见小说元素，可自定义扩展

## 使用方法

### 1. 基础功能（自动高亮）
插件会自动在生成的文本中查找预设的特殊名词，并添加【】框。例如：
- 阵营：天剑宗 → 【天剑宗】
- 功法：九转金丹诀 → 【九转金丹诀】
- 物品：上古神器 → 【上古神器】

### 2. 管理高亮名词（通过工具）
LLM可以通过调用以下工具来管理高亮名词：

#### 添加名词
```python
add_highlight_term(category="factions", term="新阵营名称")
```

#### 移除名词
```python
remove_highlight_term(category="cultivation_methods", term="旧功法名称")
```

#### 查看当前名词列表
```python
list_highlight_terms()
```

## 初始预设名词
- **阵营（factions）**：天剑宗、魔教、圣光教会、暗影议会、龙族、精灵族
- **功法（cultivation_methods）**：九转金丹诀、魔神功、圣光术、暗影步、龙象般若功、御剑术
- **特殊物品（special_items）**：上古神器、灵丹妙药、秘籍、法宝

## 技术实现
- 使用 `on_before_scene_write` 钩子在LLM生成文本前注入高亮提示
- 通过 `workspace.safe_read_json` 和 `workspace.safe_write_json` 安全存储高亮名词配置
- 提供三个工具接口供LLM动态管理高亮名词

## 注意事项
- 高亮功能仅在LLM生成文本时生效
- 名词匹配是精确匹配，确保输入的名词准确
- 支持中英文混合的名词高亮
